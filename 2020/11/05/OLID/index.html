OLID: Offensive Language Identification Dataset

OLID는 Offensive Language Identification Dataset의 약어로, Semeval-2019 Task 6: OffensEval의 Dataset으로 사용되었다.

Motivation

OLID 전에 공개된 데이터셋은 흔히 우리가 악성댓글이라고 말하는 텍스트의 일부 특징만을 어노테이션해서 공개하였다. 다시 말해, 이전 연구는 Aggression identification, Bullying detection, Hate speech identification, Offensive language, Toxic comments 각각의 aspect에만 집중되어 있다.

Main Contributions

그래서 저자들은 악성댓글의 전반적인 특성을 모두 반영하기 위해 A, B, C 3개의 계층구조 태깅 방식을 활용하였다.

  • A: Offensive Language Detection → NOT (not offensive) / OFF (offensive)
  • B: Categorization of Offensive language → TIN (target insult) / UNT (untargeted)
  • C: Offensive language Target Identification → IND (individual) / GRP (group) / OTH (other)

총 14,100개의 영어 트위터 데이터이고, highly imbalanced 되어 있다.
