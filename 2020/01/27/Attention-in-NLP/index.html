Attention in NLP

You can't cram the meaning of a whole %&!# sentence into a single &!#* vector!

Raymond Mooney

AttentionмқҖ single vectorм—җ н•ң л¬ёмһҘмқҳ мқҳлҜёлҘј мҷ„лІҪн•ҳкІҢ лӢҙмқ„ мҲҳ м—Ҷкё° л•Ңл¬ём—җ н•„мҡ”н•ң мҲңк°„м—җ, н•„мҡ”н•ң м •ліҙлҘј мӮ¬мҡ©н•ҳкё° мң„н•ң л°©лІ•мқҙлӢӨ. кё°ліём ҒмңјлЎң query vectorмҷҖ key vectorмқҳ мЎ°н•©мңјлЎң attention weightк°Җ кі„мӮ°лҗңлӢӨ. м—¬кё°м„ң "мЎ°н•©"мқҳ л°©лІ•м—җлҠ” нҒ¬кІҢ л‘җк°Җм§Җк°Җ мһҲлӢӨ. н•ҳлӮҳлҠ” Additive AttentionмңјлЎң query vectorмҷҖ key vectorм—җ feed-forward networkлҘј м Ғмҡ©н•ң кІғмқҙкі , лӢӨлҘё н•ҳлӮҳлҠ” Dot-Product AttentionмңјлЎң л¬ёмһҗк·ёлҢҖлЎң query vectorмҷҖ key vectorмқҳ dot-productлҘј мқҙмҡ©н•ң кІғмқҙлӢӨ. мқҙлІҲ кёҖм—җм„ңлҠ” к°Ғ Attention л°©лІ•л“Өкіј мқҙл“Өмқҳ мһҘлӢЁм җмқ„ мҶҢк°ңн•ҳл Өкі  н•ңлӢӨ.

Additive Attention

Additive AttentionмқҖ query vectorмҷҖ key vectorмқҳ мЎ°н•©мңјлЎң attention к°’мқ„ м–»мқ„ л•Ң лӢЁмқј hidden layerлҘј к°Җм§„ feed-forward networkлҘј мқҙмҡ©н•ңлӢӨ. query vector (q) мҷҖ key vector (k) к°Җ к°ҷмқҖ dimensionмқ„ к°Җм§Ҳ н•„мҡ”к°Җ м—Ҷмңјл©°, dimensionмқҳ нҒ¬кё°м—җ мғҒкҙҖм—Ҷмқҙ мўӢмқҖ м„ұлҠҘмқ„ ліҙмқёлӢӨлҠ” мһҘм җмқҙ мһҲлӢӨ.

Bilinear[2]

\[a(q, k)= q^{T} Wk\]

\(q\) мҷҖ \(k\) м—җ \(W\) лҘј кіұн•ҳлҠ” л°©лІ•мқҙлӢӨ. \(q\) лҘј linear transform мӢңнӮЁ нӣ„, \(k\) мҷҖ dot-productлҘј н•ң кІғкіј к°ҷлӢӨ.

Multi-layer Perceptron[3]

\[a(\boldsymbol{q}, \boldsymbol{k})=\boldsymbol{w}_{2}^{\top} \tanh \left(W_{1}[\boldsymbol{q} ; \boldsymbol{k}]\right)\]

\(q\) мҷҖ \(k\) лҘј concatмӢңнӮЁ нӣ„ single hidden layerмҷҖ activation н•ЁмҲҳлЎң tanhлҘј мӮ¬мҡ©н•ң feed-forward networkлҘј мӮ¬мҡ©н•ҳмҳҖлӢӨ. мқҙ л°©лІ•мқҖ мқөмҲҷн•  кІғмқҙлӢӨ. мҷңлғҗн•ҳл©ҙ Neural Machine Translation by Jointly Learning to Align and Translateм—җм„ң мқҙ attentionмқ„ мқҙмҡ©н–Ҳкё° л•Ңл¬ёмқҙлӢӨ. м•„лһҳмқҳ мҳӨлҘёмӘҪ мҲҳмӢқм—җм„ң \(a\)лҠ” alignment modelлЎң мң„м—җм„ң м–ёкёүн•ң feed-forward networkмҷҖ к°ҷмқҖ м—ӯн• мқ„ н•ңлӢӨ.

Dot-Product Attention

Dot-Product attentionмқҖ \(^{} \)мқ„ кё°л°ҳмңјлЎң attention weightлҘј кө¬н•ҳлҠ” л°©лІ•мқҙлӢӨ. Additive attentionкіјлҠ” лӢ¬лҰ¬ hidden layerлҘј кіұн•ҳлҠ” кіјм •мқҙ м¶”к°Җлҗҳм§Җ м•Ҡм•„м„ң м—°мӮ° мҶҚлҸ„мҷҖ space мёЎл©ҙм—җм„ң нҡЁмңЁм ҒмқҙлӢӨ. н•ҳм§Җл§Ң л°ҳл“ңмӢң \(q\)мҷҖ \(k\)мқҳ dimensionмқҙ к°ҷм•„м•ј н•ңлӢӨлҠ” м ңм•ҪмЎ°кұҙмқҙ мһҲмңјл©°, dimensionмқҙ нҒҙ л•Ң н•ҷмҠөм—җ л°©н•ҙк°Җ лҗңлӢӨлҠ” лӢЁм җмқҙ мһҲлӢӨ.

Dot-Product

\[a(\boldsymbol{q}, \boldsymbol{k})=\boldsymbol{q}^{\top} \boldsymbol{k}\]

\(q\)мҷҖ \(k\)мқҳ elment-wise productмқҳ н•©мқҙлӢӨ. мқҙ м—°мӮ°мқҳ нҠ№м„ұ мғҒ, л°ҳл“ңмӢң dimensionмқҙ к°ҷм•„м•ј н•ңлӢӨ.

л§Ңм•Ҫ \(q\)мҷҖ \(k\)мқҳ к°Ғ мҡ”мҶҢк°Җ лҸ…лҰҪм Ғмқҙкі  нҸүк· мқҙ 0, л¶„мӮ°мқҙ 1 мқё л¶„нҸ¬мқҳ random variableмқҙлқјл©ҙ, \({q}^{\top}{k}\) лҠ” нҸүк· мқҙ 0мқҙкі  л¶„мӮ°мқҙ dimensionмқҳ нҒ¬кё°мқё л¶„нҸ¬лҘј л”°лҘёлӢӨ. л¶„мӮ°мқҙ мҰқк°Җлҗҳл©ҙ dot-productм—җ softmaxлҘј м·Ён–Ҳмқ„ л•Ң м–ҙл–Ө к°’мқҖ көүмһҘнһҲ нҒ¬м§Җл§Ң лҢҖл¶Җл¶„мқҳ к°’мқҖ көүмһҘнһҲ мһ‘кІҢ л§Ңл“ лӢӨ. мһ‘мқҖ к°’л“ӨмқҖ back-propagation мӢң gradientлҸ„ мһ‘кё° л•Ңл¬ём—җ м „мІҙм ҒмңјлЎң н•ҷмҠөмқҙ мһҳ лҗҳм§Җ м•ҠкІҢ л§Ңл“ лӢӨ. л”°лқјм„ң dimensionмқҙ нҒ° кІҪмҡ° м„ұлҠҘмқҙ мўӢм§Җ м•ҠлӢӨ.

Scaled Dot-Product[4]

\[a(\boldsymbol{q}, \boldsymbol{k})=\frac{\boldsymbol{q}^{\top} \boldsymbol{k}}{\sqrt{|\boldsymbol{d}|}}\]

Scaled Dot-ProductлҠ” Attention is All You Need м—җм„ң мІҳмқҢмңјлЎң мҶҢк°ңлҗң л°©лІ•мқҙлӢӨ. Dot-Productмқҳ dimensionмқҙ нҒҙ л•Ң н•ҷмҠөмқҙ мһҳ лҗҳм§Җ м•ҠлҠ” лӢЁм җмқ„ к·№ліөн•ҳкІҢ мң„н•ҙ dot-product кІ°кіјлҘј \(q\)мқҳ dimension \(d\) (\(k\) мқҳ dimension мқҙкё°лҸ„ н•ҳлӢӨ) мқҳ root к°’мңјлЎң лӮҳлҲ„м–ҙмЈјм—ҲлӢӨ.[5]

Conclusions

Additive attentionмқҖ dimensionм—җ мғҒкҙҖм—Ҷмқҙ мўӢмқҖ кІ°кіјлҘј лӮҙкі , attentionмқ„ кі„мӮ°н•ҳлҠ” мһ¬лЈҢмқё query vectorмҷҖ key vectorмқҳ dimensionм—җ мғҒкҙҖм—Ҷмқҙ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ. н•ҳм§Җл§Ң мөңк·јмқҳ NLP trendлқјкі  н•  мҲҳ мһҲлҠ”, лҢҖлҹүмқҳ лҚ°мқҙн„°лҘј көүмһҘнһҲ нҒ° лӘЁлҚёлЎң н•ҷмҠөмӢңнӮӨлҠ” л°©лІ•м—җлҠ” м•Ҳк·ёлһҳлҸ„ л§ҺмқҖ м—°мӮ°лҹүм—җ л¶ҖлӢҙмқҙ лҗҳлҠ” л°©лІ•мқҙлӢӨ. к·ёлһҳм„ң кі„мӮ°мқҳ л¶ҖлӢҙмқҙ м Ғмңјл©ҙм„ң dimensionмқҙ нҒ° кІҪмҡ°м—җ лҢҖн•ҙм„ңлҸ„ мўӢмқҖ м„ұлҠҘмқ„ ліҙмқҙлҠ” scaled dot-product кё°л°ҳмқҳ attentionмқҙ мһҳ м“°мқҙлҠ” нҺёмқҙлӢӨ.
